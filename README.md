# Urban Mobility & Bike Sharing Analytics Dashboard

Behavioral, Operational & Geospatial Analysis of BIXI Montréal Bike Sharing System using Google Looker Studio.

---

## Project Overview

This project analyzes urban mobility and bike-sharing usage patterns using the **BIXI Montréal Bike Sharing Trips (2021)** dataset.

The dashboard transforms raw transportation trip data into interactive business intelligence insights to help understand:

- Rider behavior
- Operational demand
- Peak riding activity
- Station utilization
- Ride duration patterns
- Geospatial mobility concentration

The project was developed using **Google Looker Studio (Data Studio)** for dashboard visualization and analytics storytelling.

---

## Business Problem

Urban bike-sharing systems generate large amounts of trip data daily. However, transportation operators and city planners often struggle to identify:

- Peak congestion periods
- High-demand stations
- Rider behavioral trends
- Member vs casual rider patterns
- Operational bottlenecks
- Mobility demand concentration

This dashboard solves these challenges through interactive analytics and visual storytelling.

# Data Cleaning & Transformation

The dataset was preprocessed before dashboard development.

### Cleaning Steps
- Removed duplicate records
- Handled missing values
- Converted timestamps into datetime format
- Standardized station names
- Validated ride duration values

### Derived Features
- Ride Category
- Peak Hour Classification
- Weekday Analysis
- Hourly Activity Groups
- Rider Type Segmentation

---

# Key Insights

- Members contribute approximately **84%** of total rides.
- Peak riding activity occurs between **7 AM and 9 AM**.
- Medium rides dominate overall trip distribution.
- Metro-connected stations show significantly higher activity.
- Weekend rides indicate strong recreational usage patterns.
- Mobility demand is geographically concentrated.

---

# Business Recommendations

- Optimize bike redistribution during commute hours.
- Prioritize high-demand stations for operational monitoring.
- Improve commuter-focused services.
- Introduce member retention and loyalty programs.
- Expand stations in dense mobility corridors.
